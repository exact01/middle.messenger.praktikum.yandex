## Описание

Вёрстка макета Яндекс практикума по курсу middleFrontEndDeveloper!

## Установка

- `npm install` — установка стабильной версии,
- `npm run dev` — запуск версии для разработчика,
- `npm run start` — сборка стабильной версии.
